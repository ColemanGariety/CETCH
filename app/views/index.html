What is this?

CETCH is a computer science competition involving participants from all over the world. Each week we solve a computer science puzzle of varying complexity and upload our programs. On Friday at midnight, all the entries are run in a sandbox and the execution times are clocked. The fastest solution wins. A new competition begins immediately after the previous one has ended.

Rules:

  1. You must use one of the languages specified
  2. You must only use the standard libraries for that language
  3. Your program must only use the information presented in the problem
  4. Fastest execution time wins
