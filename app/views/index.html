What is this?

CETCH is a computer science competition involving coders all over the world. Each week we solve a computer science puzzle of varying complexity and upload the solution. On Friday at midnight, all the entries are run in a sandbox and the execution times are clocked. The program with the fastest execution time wins.

Rules:

  1. Your solution must employ one of the allowed languages
  2. Your solution must use only that language's standard libraries
  3. Your solution must be derived exclusively from the problem statement
  4. The fastest execution time wins
